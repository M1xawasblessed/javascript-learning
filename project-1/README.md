## Project 1: Smart Supermarket Cart

**Description:**
A mini-project to simulate a shopping cart logic for a local supermarket, utilizing modern JavaScript data structures.

**Technologies & Concepts Used:**
- **Array Methods:** Used `.find()` to locate items in the inventory and logic to iterate over data.
- **Map:** Used to store cart items as `key-value` pairs (Item Name -> Quantity), dynamically updating quantities if the item already exists.
- **Set:** Used to keep track of unique product categories added to the cart, automatically preventing duplicates.
- **Functions:** Encapsulated logic for adding items and calculating the total price.
