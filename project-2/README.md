## Project 2: Spanish Vocabulary Flashcards

**Description:**
An interactive flashcard application built to help practice Spanish vocabulary. The app displays a Spanish word, checks the user's English translation, and persists the user's score.

**Technologies & Concepts Used:**
- **DOM Manipulation:** Dynamically created the user interface elements (headings, inputs, buttons) directly from JavaScript.
- **Event Listeners:** Used the `click` event to capture and validate user input against the correct translation.
- **Web Storage API:** Implemented `localStorage` to save the user's score so progress is not lost upon page reload.
- **Array of Objects:** Stored vocabulary pairs efficiently to cycle through the test.
