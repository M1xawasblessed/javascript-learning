## Day 19: Web Storage API

**Today's Progress:**
- Learned how to store data in the browser using the Web Storage API.
- Used `localStorage` to save data that persists even after the browser is closed.
- Used `sessionStorage` to save data that is cleared when the page session ends.
- Practiced saving, retrieving, and deleting specific items using `.setItem()`, `.getItem()`, and `.removeItem()`.
- Learned how to store complex data structures (like objects) by converting them to strings with `JSON.stringify()` and back to objects with `JSON.parse()`.
