## Project 3: Transport Zone Ticket Validator

**Description:**
A simulation of a public transport validation system (like the T-usual card). It checks if a user's ticket is valid for a specific station based on zone restrictions.

**Technologies & Concepts Used:**
- **Asynchronous JavaScript:** Used `async/await` and `Promises` to simulate fetching data from a remote server.
- **Error Handling:** Implemented `try...catch...finally` blocks to handle invalid station requests gracefully.
- **Array Methods:** Utilized `.find()` and `.includes()` for matching station names and verifying valid zones.
