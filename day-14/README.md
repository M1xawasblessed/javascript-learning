## Day 14: Maps and Sets

**Today's Progress:**
- Learned about `Set`, a data structure that stores a collection of unique values, automatically removing duplicates.
- Explored `Map`, a collection of keyed data items, similar to an Object but allowing keys of any type (including objects and numbers).
- Practiced built-in methods like `.add()` and `.delete()` for Sets, and `.set()`, `.get()`, and `.has()` for Maps.
- Iterated over Map elements using the `.forEach()` method.
